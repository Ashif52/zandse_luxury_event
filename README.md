# ✦ ZANDSE | Luxury Event Planning & Breathtaking Experiences ✦

Welcome to **ZANDSE**, a premium, cinematic web experience designed for a high-end luxury event planning and management brand. This application features state-of-the-art interactive front-end designs, fluid animations, custom animations, and a rich, gold-and-black aesthetic tailored to wow clients from first glance.

---

## ✦ Core Features

- **Cinematic Visual Identity**: A rich HSL gold-and-black color palette, premium typography (Cinzel & Poppins), smooth gradients, and elegant borders.
- **Golden Dust Particle System**: Immersive custom canvas-driven particle generators flowing in the background of the hero sections and footer.
- **Bespoke Custom Cursor**: A smooth, lag-free cursor follower that reacts dynamically when hovering over cards, buttons, and links.
- **3D Card Tilt Effect**: Interactive cards (services and packages) that tilt in 3D perspective based on the mouse hover position.
- **Nostalgic Timeline Tracker**: A scroll-driven, visual progress bar that charts our custom service timeline.
- **Horizontal Scroll Carousel**: Smooth, draggable, and scroll-wheel friendly categories for service page packages.
- **Lightbox Visual Galleries**: Interactive media grids that pop up in a gorgeous glassmorphism lightbox on image clicks.
- **Testimonial Auto-Slider**: Fluidly transitioning reviews carousel.
- **Fully Responsive**: Flawlessly responsive across mobile devices, tablets, and wide monitors.
- **SEO & Search Optimization**: Done to the highest standards, featuring exact canonical metadata, Open Graph (OG) tags, and Google-readable JSON-LD schemas.

---

## ✦ Technology Stack

- **Core Structure**: HTML5 (Semantic elements)
- **Styling & Layout**: CSS3 (Vanilla CSS variables, Flexbox, Grid, custom keyframes, and transitions)
- **Interactive Logic**: Vanilla JavaScript (ES6+, IntersectionObserver, Canvas API, device detection)

## ✦ Getting Started (Local Development)

Since this project runs entirely on vanilla client-side technologies, you do not need any installation or compile steps:

1. Clone or download this repository:
   ```bash
   git clone <YOUR_GITHUB_REPOSITORY_URL>
   ```
2. Navigate into the folder:
   ```bash
   cd zandse_luxury_event
   ```
3. Open `index.html` directly in your web browser, or use a local development server (such as VSCode's **Live Server** extension or `npx serve`) to serve the assets properly.

---

## ✦ Deployment

This website is static-site ready and can be hosted for free on:
- **GitHub Pages**: Simply push to your repository, enable Pages under settings, and point to the root directory.
- **Vercel / Netlify**: Connect the GitHub repository directly for zero-config auto-deployments on every commit.
